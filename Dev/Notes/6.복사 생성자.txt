1. 얕은 복사 vs 깊은 복사 
    1) 비유 
        -           개집(원복)                  개집(복사본)
                --------------------         ----------------
                  이블, 개껌, 밥그릇           이블, 개껌, 밥그릇

    2) 얕은 복사 (shallow copy)       
        - 객체 복사 시, 객체의 멤버를 1:1로 복사 
        - 객체의 멤버 변수에 동적 메모리가 할당된 경우 
            - 복사본은 원본 객체가 할당 받은 메모리를 공유하는 상태임 

    3) 깊은 복사 (deep copy)
        - 객체 복사 시, 객체의 멤버를 1:1로 복사 
        - 객체의 멤버 변수에 동적 메모리가 할당된 경우 
            - 복사본은 원본이 가진 메모리 크기 만큼 별도로 동적 할당 
            - 원본의 동적 메모리에 있는 내용을 복사본에 복사 
        - 완전한 행태의 복사 
            - 복사본과 원본은 메모리를 공유하지 않음                                   

2. 복사 생성자 (copy constructor)
    1) 객체의 복사 생성시 호출되는 특별한 생성자 
    2) 특징
        - 한 클래스에 오직 한 개만 선언 가능 
    3) 모양
        - 클래스에 대한 참조 매개변수를 가지는 독특한 생성자 
        - 예)   class Circle {
                    Circle(const Circle& c) {}
                }
                                 
3. 디폴트 복사 생성자 
    1) 복사 생성자가 선언되어 있지 않는 클래스 
        - 컴파일러는 자동으로 디폴트 복사 생성자 삽입 
    2) 예:
            class Circle {
                int radius;
            public:
                Circle(int r);
                double getArea();                
            }               

            Circle src(30);	
            Circle dest(src);       //복사 생성


            Circle(const Circle& c) {
                // 컴파일러가 삽입하는 디폴트 복사 생성자
            }                          

4. C++ 컴파일러가 만들어 주는 함수 (자동으로 생성)
    1) 기본 생성자 
    2) 소멸자
    3) 기본 복사 생성자           
        - 얕은 복사
