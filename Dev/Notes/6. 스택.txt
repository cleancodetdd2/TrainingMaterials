1. 스택(Stack)
    1) 한쪽으로 들어가서 한쪽으로 나오는 자료 구조임.
    2) 수식 계산 등 알고리즘 다방면으로 활용됨
    3) 용어
        - PUSH : 스택에 데이터를 넣음
        - POP : 스택에서 데이터를 빼냄 
    4) Last In First Out (LIFO) 방식으로 작동함 
        - 가장 나중에 들어온 데이터가 가장 먼저 나가는 구조           

2. 특징
    1) LIFO 구조 
    2) 제한된 접근 
        - 스택의 데이터는 오직 상단 요소에만 접근할 수 있음.
    3) 메모리 관리 효율성이 좋음
        - 함수 호출 시 지역 변수를 저장하는 데 자주 사용 
    4) 사용 예
        - 함수 호출 관리
            - 함수 호출 시 각 함수의 상태를 저장하고,
              함수 종료되면 이전 상태로 돌아가기 위해 스택을 사용함 
        - 수식 계산
            - 후위 표기법 계산 등에 사용
        - Undo 기능
            - 이전 상태로 돌아가는 기능 구현                                    

3. 스택의 기본 연산
    1) Push 
    2) Pop 
    3) Top (Peek)
        - 스택의 가장 상단에 있는 데이터를 반환하되, 제거하지는 않은 연산
    4) IsEmpty
        - 스택이 비어 있는지를 확인하는 연산        

4. 스택의 구현
    1) 배열을 이용한 구현 방법과 연결 리스트를 이용한 구현 방법으로 나누어짐
    2) 가장 기본적인 형태의 자료구조임
