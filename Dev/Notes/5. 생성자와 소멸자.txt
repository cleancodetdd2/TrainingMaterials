1. C++ 생성자(Constructor)
    1) 객체를 생성함과 동시에 멤버 변수를 초기화 함.
    2) 생성자는 특별한 메서드, 함수라고 할수 있음.
        - 클래스의 이름과 동일한 함수명으로 구현됨.
        - 생성자는 반환 값이 없음.
        - 여러 번 정의되어 다양한 방법으로 객체를 초기화할 수 있음.
            - 생성자 오버로딩

2. 생성자와 소멸자 (Constructor & Destructor)
                       Constructor                      Destructor
    ------------------------------------------------------------------------                       
    1) 호출 시점        객체 생성 시                      객체 소멸 시
    2) 함수명           클래스명                          ~클래스명
    3) 리턴값           없음                              없음
    4) 매개변수         가질수 있음                         없음
    5) 활용             instanace variable 초기화시        동적 메모리 해제 시           

3. 생성자 종류
    1) 기본 생성자(default constructor)
        - 별도로 생성자를 구현하지 않으면 기본 생성자가 사용됨 
        - 매개변수를 가지지 않으며 멤버변수는 0, NULL등의 값으로 초기화됨 
    2) 복사 생성자(Copy constructor)  
        - 다른 인스턴스의 참조(Reference)를 인수로 받아서,
          그 참조를 이용해 자신의 인스턴스를 초기화할 수 있도록 해줌.       
        - 깊은 복사(Deep copy)를 이용해 만들어진 인스턴스는 
          기존의 인스턴스와 다른 메모리 공간에 할당되어 독립적임.  

4. 생성자로 초기화하는 방법
    1) 일반적인 방법 
        - MyClass(int _a, double _d, char _c)
          {
            a = _a; d = _d; c = _c;
          }    
    2) 클론 (:)을 이용한 초기화 방법
        - MyClass(int _a, double _d, char _c) : a(_a),d(_d),c(_c)   
          {

          }     
    3) 생성자에서 다른 생성자를 호출할 수 있음
        - MyClass() : MyClass(3, 3.6,'A')       <---- 위임 생성자
          {

          }     
        - MyClass(int _a, double _d, char _c)   <---- 타겟 생성자 
          {

          }                 

5. 소멸자 (Destructor)
    1) 객체의 수명이 끝났을 때 객체를 제거하기 위한 목적으로 사용됨.
    2) 객체의 수명이 끝났을 때 자동으로 컴파일러가 소멸자 함수를 호출함.
    3) 소멸자 이름은 클래스 이름과 동일하며 물결 기호(~)를 이용해 정의함
