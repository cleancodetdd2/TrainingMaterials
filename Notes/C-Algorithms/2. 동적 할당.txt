1. 메모리 
    1) 프로그램이 실행되는 동안 저장되는 데이터는 RAM 메모리 공간에 저장됨 
    2) 영역(Area)
        - Stack
            - 함수 내에서 선언된 지역 변수 (매개변수) 
        - heap
            - 동적으로 할당된 메모리 
            - 프로그래머 할당/해제 
        - Data 
            - 전역 변수나 정적변수(static 변수)
            - 프로그램 실행~종료될 때까지 메모리 할당

2. 동적 메모리 할당
    1) intro 
        - 일반적으로 C언어에서 배열의 경우 사전에 적절한 크기만큼 할당해
          주어야 함.
        - 우리가 원하는 만큼만 메모리를 할당해서 사용하고자 한다면
          동적 메모리 할당을 사용함 
        - 동적이라는 말의 의미는 '프로그램 실행 도중에'라는 의미임.

    2) 동적 메모리 할당 함수
        - malloc(할당할 바이트 크기);
        - 원하는 만큼의 메모리 공간을 확보할 수 있음.
        - <stdlib.h> 라이브러리에 정의 됨 
        - 동적 메모리 할당은 수행할 때마다 포인터의 주소는 달라짐.
        - 동적으로 할당된 변수는 <힙 영역>에 저장됨            

        - 동적으로 할당된 변수는 반드시 free() 함수로 메모리 해제 해주어야 함 
            - 메모리 해제를 않하면 언젠가는 오류가 발생함
            - 메모리 누수(Memory Leak) 방지는 개발자의 핵심 역량임.
