1. 자료 구조의 개요 
    1) 자료 구조의 필요성
        - 데이터를 효과적으로 저장하고, 처리하는 방법
        - 메모리 양과 시간 
        - 원하는 데이터를 가장 빠르게 찾도록 도와주는 자료구조 

    2) 기본적인 자료구조들 
        - 선형 구조 
            - 배열 
            - 연결 리스트 
            - 스택
            - 큐

        - 비선형 구조   
            - 트리 
            - 그래프       

2. 연결 리스트 (Linked List)
    1) 연결된 노드에 데이터를 저장하는 자료 구조임. 
    2) 각 노드는 데이터와 다음 노드를 가리키는 포인터를 가짐  
    3) 데이터를 동적으로 추가, 삭제할 수 있음 

3. 연결 리스트의 구성요소 
    1) 헤드(head) : 연결 리스트에서 첫번째 노드의 주소를 저장한 포인터    
    2) 노드(node) : 연결 리스트의 요소 
        - 값(value) : 노드에 저장된 데이터 
        - 링크(link) : 다른 노드와의 연결을 위한 포인터 

4. 연결 리스트의 종류 
    1) 단일 연결 리스트 (Singly Linked List)          
        - 각 노드가 다음 노드에 대한 참조만 가짐.
    2) 이중 연결 리스트 (Doubly Linked List)    
        - 각 노드가 이전노드와 다음노드에 대한 참조를 모두 가짐 
    3) 환형 연결 리스트 (Circular Linked List)     
        - 마지막 노드가 첫번째 노드를 가리킴 
        - 리스트의 끝과 처음이 연결된 형태임
