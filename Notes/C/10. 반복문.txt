1. for 반복문       ①②③④⑤⑥⑦⑧⑨
    1) 반복 횟수, 반복 및 정지 조건을 자유 자재로 제어 가능 
    2) loop statement 
              ①      ②       ④
        for(초기식; 조건식; 변화식)
        {
            반복할 코드 ③                 
        }
            - 초기식 => 조건식 => 루프 (반복할 코드) => 변화식 => 조건식 =>...
                ①        ②            ③                ④         ② 
    3) 실행 연습
        - 초깃값을 1부터 시작하는 경우                 
        - 초깃값을 감소시키는 경우 

    4) for 반복문은 if조건문과 마찬가지로 끝에 ;을 붙이면 안됨 
    
    5) for 반복문으로 반복할 코드가 한 줄이면 중괄호 생략 가능 
        - 두 줄 이상일 때는 중괄호 생략에 주의 

    6) for 반복문에서 초기화 변수 두개 사용 가능,
       변화식에서도 ++, -- 뿐만 아니라 +=, -=, *=, /=도 사용 가능 

    7) for 반복문으로 무한 루프 만들기 
        - 초깃값, 조건식, 변화식을 모두 생략하면 무한 루프 

2. while 반복문 
    1) loop statement 형식

        초기식
        while(조건식)
        {
            반복할 코드
            변화식
        }

            - 초기식 => 조건식 => 반복할 코드 및 변화식 => 조건식....
    2) 무한 루프
        - while에 1 혹은 불값인 true를 넣으면 됨             

3. do while 반복문 
    1) 최소 한 번은 실행함 
    2) loop statement 형식

        초기식
        do
        {
            반복할 코드
            변화식ㄴ
        } while(조건식);

4. 반복 횟수가 정해지지 않은 경우 

5. break, continue로 반복문 제어하기 
    1) break 
        - for, while, do while, switch 구문에서 제어흐름를 벗어나기 위해 사용
    2) continue로 코드 실행 건너뛰기
