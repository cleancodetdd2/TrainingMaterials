1. 개념
    1) 표준 입출력 함수는 모니터, 키보드를 통해 데이터를 입출력함
    2) 파일입출력 함수 
        - 보조기억장치에 파일(file)이라는 개념을 가지고 데이터를 입출력함 
    3) 파일 스트림 (file stream)
        - 모든 입력과 출력은 스트림이라고 하는 공통된 인터페이스를 사용하여
          파일로 취급되는 장치들과 연결됨.        

2. 파일포인터를 활용한 파일 IO         
    1) 단계
        - 파일 열기
        - 입출력 수행하기
        - 파일 닫기 
    2) 제공하는 프로그램 
        - FILE 구조체 
        - 파일 사용 모드 
            - r (읽기 전용)
            - w (쓰기 전용)
            - a (추가 전용)
        - fputc()
            - 스트림에 한 문자를 출력하는 함수
            - 에러가 발생할 경우 EOF(-1)를 리턴함 
        - fgetc()
            - 스트림으로부터 한 문자를 읽어오는 함수 
            - 문자를 들인 문자를 리턴

        - fscanf()
            - 파일에서 서식화된 자료를 입력하기 위해 사용되는 함수
